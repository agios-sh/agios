When running `agios init`, automatically set up the SessionStart hook for the user's agent so that `agios` runs at session start.

**Initial scope:** Claude Code (configure `.claude/settings.json` hooks).

**Future:** Support Codex and other agents as well.

**Key detail:** The hook should use the same binary path that was invoked for `agios init` (i.e. resolve the running binary's path and persist that into the hook config).
